# [Awesome CTF Book](https://book.ph0en1x.com/)

## 这是什么

这是一本关于CTF的小书。

## 为什么要写这本书

* 进行知识整理。
* 使读者更好地了解CTF。
* 为想在网络安全领域发展但比较迷茫的朋友提供一个方向。
* 帮助读者绕过一些我们踩过的坑。
* 输出是一种很好的学习方法。

## 目标与期望

希望能通过这本书，以及配套的资源，能使读者真正有所收获，吸引更多的人加入到我们当中来。愿一同进步！
